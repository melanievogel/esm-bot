Telegram ESM-bot

This bot program conducts a 5-day signal-contigent sampling Experience Sampling Study.

Getting Started

For starting the program, execute the bot.js file via the command node bot.js from the terminal. As prefered IDE use Visual Studio Code with integrated terminal.

Prerequisites

First, create a new bot token via the botfather in Telegram and paste it into the token variable.
Install the newest verson of mongodb and nodejs. The node module mongoose was used for database operations.

MongoDB must be running before starting bot.js.

Deployment

The application can be deployed on linux server via screen. Other possible solutions include platforms such as Heroku or Amazon Web Services.

May be useful for AWS deployment:
https://claudiajs.com/
